# STS 2 Strategizer

## What This Is

A Progressive Web App for Slay the Spire 2 players to look up cards and discover synergies mid-run. Users search for a card by name, see full card details pulled from Mobalytics, and get algorithmically-generated synergy suggestions with full combo breakdowns — scoped to their current character's card pool.

## Core Value

Fast card lookup + synergy discovery during an active run, scoped to your character's pool.

## Context

- **Data source:** https://mobalytics.gg/slay-the-spire-2/wiki/cards — scraped/fetched and used as source of truth for all card data
- **Use case:** Mid-run reference — speed of lookup matters; UI should be low-friction
- **Character model:** STS2 has multiple playable characters with distinct card pools; the app defaults to a selected character and filters synergies accordingly
- **Synergy engine:** Rule-based — detects synergies from card keywords and mechanics (e.g. cards that both scale with Strength, cards that combo with Exhaust, etc.)
- **Combo format:** Full breakdown per synergy pair — interaction explanation, prioritization advice, value rating (format TBD by implementation)
- **Data freshness:** Periodic auto-refresh to keep up with game patches

## Constraints

- **Data:** Must respect Mobalytics as source of truth — no inventing card data
- **Platform:** PWA (web-first, installable) — no native app build required
- **Connectivity:** Light offline support (PWA shell), but full functionality requires connection
